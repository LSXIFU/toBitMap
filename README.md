# 图像/视频转C语言数组位图工具

一个用于将图像和视频转换为C语言数组位图的Web工具，支持多种转换模式和自定义设置。

## 功能特性

### 🖼️ 图片转换
- 支持单张图片上传
- 支持批量处理（ZIP压缩包）
- 自定义目标尺寸
- 多种阈值类型（RGB平均值、亮度、RGB独立调整）
- 支持框选裁剪区域
- 锁定比例功能

### 🎬 视频转换
- 支持视频文件上传
- 自定义帧率选择
- 自动分割为多个压缩包
- 去重帧功能
- 支持框选裁剪区域

### ✏️ 手绘模式
- 内置手绘画板
- 自定义画布大小
- 笔触大小调整
- 黑白反转功能
- 实时预览

### ⚙️ 导出设置
- 自定义文件名
- 自定义数组名
- 自定义宽高变量名
- 一键导出C语言头文件

## 使用方法

### 在线使用
直接打开 `index.html` 文件即可在浏览器中使用。

### 本地运行
```bash
# 使用Python启动本地服务器
python -m http.server 8000

# 或使用Node.js
npx serve .
```

然后在浏览器中访问 `http://localhost:8000`
